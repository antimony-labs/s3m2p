# CRM Tool

Customer Relationship Management (CRM) utility for Antimony Labs.

## Overview
A WebAssembly-based CRM tool designed to manage customer interactions and data locally within the browser.

## Features
- **Local Storage**: Uses the browser's `localStorage` for data persistence.
- **WASM Performance**: Rust-based logic for data processing.
- **Frontend Integration**: Designed to work with a lightweight HTML/JS frontend.

## Dependencies
- `wasm-bindgen`
- `web-sys`
- `serde`
