Buck Converter (Step-Down)

Steps voltage down using high-frequency switching.

Key equations:
D = Vout / Vin
L = Vout(1-D) / (fsw × ΔIL)

Operation: When switch ON, energy stored in inductor. When OFF, diode conducts and inductor releases energy to load.
Boost Converter (Step-Up)

Steps voltage up using inductor energy storage.

Key equations:
D = 1 - Vin / Vout
L = Vin × D / (fsw × ΔIL)

Operation: When switch ON, inductor charges from input. When OFF, inductor + input voltage combine to boost output.
LDO Regulator (Linear)

Linear dropout regulator - no switching noise.

Key equations:
Pdiss = (Vin - Vout) × Iout
η = Vout / Vin

Trade-offs: Low noise, simple design, but wastes power as heat. Best when Vin ≈ Vout.
