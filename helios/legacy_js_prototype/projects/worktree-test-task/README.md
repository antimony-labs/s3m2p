# Worktree Test Task

This is a test project created to verify the worktree automation functionality.

## Purpose

Test that the worktree automation runs correctly when a new worktree is created. The automation should execute `npm install` as configured in `.cursor/worktrees.json`.

## Testing

To test the worktree automation:

1. Create a new git worktree pointing to this project directory
2. Verify that `npm install` runs automatically
3. Confirm the setup completes successfully
